# 06 — Source Code Structure

This folder documents the repository structure. For the full table of folders and their purposes, see the **project root `README.md`** → **Section 6. Source Code Documentation**.

## Quick reference

| Path | Purpose |
|------|---------|
| `src/app/` | Next.js App Router (frontend pages, Payload API, admin, sitemaps) |
| `src/components/` | Reusable UI components |
| `src/collections/` | Payload collections (Pages, Posts, Media, Services, etc.) |
| `src/globals/` | Payload globals (Header, Footer, SiteSettings, page contents) |
| `src/fields/` | Custom Payload fields |
| `src/plugins/` | Payload plugins (S3, SEO, redirects, search, forms) |
| `src/utilities/` | Shared helpers |
| `src/lib/` | Email, ClickUp, translations |
| `src/Header/`, `src/Footer/` | Header/Footer config and components |
| `public/` | Static assets |
| `scripts/` | Seed, tests, migrations |
