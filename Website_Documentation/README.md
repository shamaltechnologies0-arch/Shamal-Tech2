# Website Documentation — Index

This folder is the **single place** for all corporate website documentation. Place the following files here (or link to them from here if stored in a secure/shared drive).

---

## Required documents

| # | File name | Description |
|---|-----------|-------------|
| 1 | `01_Project_Overview.pdf` or `.docx` | Website name, domain, purpose, audience, scope, launch date, version, owner, developer |
| 2 | `02_Sitemap.pdf` | Full page hierarchy / sitemap |
| 3 | `03_Technical_Documentation.pdf` | Tech stack, hosting, domain, CMS, server architecture |
| 4 | `04_Admin_Guide.pdf` | Admin URL, login, how to edit pages/posts/services, upload images |
| 5 | `05_Deployment_Guide.pdf` | Deploy steps, domain connection, env vars, build commands |
| 6 | `06_Source_Code_Structure/` | Folder (with docs inside) explaining `src/`, `public/`, etc. |
| 7 | `07_Design_System.pdf` | Colors, fonts, UI guidelines, layout, responsive rules |
| 8 | `08_SEO_Documentation.pdf` | Meta tags, keywords, sitemap.xml, robots.txt, Analytics, Search Console |
| 9 | `09_Integrations.pdf` | WhatsApp, forms, CRM, email, analytics, etc. |
| 10 | `10_Maintenance_Guide.pdf` | Backup, updates, security, troubleshooting |
| 11 | `11_Access_Credentials.xlsx` | **RESTRICTED** — Domain, hosting, CMS, analytics logins |
| 12 | `12_Assets/` | Folder for logo, images, videos, icons, fonts |

**Legal:** Document Privacy Policy, Terms & Conditions, and Cookie Policy (either here as `13_Legal.pdf` or in the Project Overview).
